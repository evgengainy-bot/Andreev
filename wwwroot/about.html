О проекте

Andreev CRM - это современная система управления клиентами, разработанная для эффективного ведения базы данных клиентов.

Возможности системы:

  • Регистрация и аутентификация пользователей с JWT токенами
  • Полный CRUD функционал для управления клиентами
  • Безопасное хранение паролей с использованием BCrypt
  • Создание резервных копий базы данных (pg_dump)
  • Доступ с любого устройства в локальной сети
  • Адаптивный дизайн для всех устройств

Технологии:

  • Backend: .NET 9.0 (C#)
  • Database: PostgreSQL 17
  • Frontend: HTML5, CSS3, JavaScript
  • Authentication: JWT Bearer
  • ORM: Entity Framework Core
